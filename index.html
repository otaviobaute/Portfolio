Nesse pequeno projeto aprendi como utilizar melhor o JavaScript para tornar uma página HTML mais interativa. O aprendizado inicial foi entender melhor como acessar os elementos que quero modificar na aplicação, no caso acessar as 'tags' e suas Id's e classes. A partir desse acesso chamar funções que modifiquem esses elementos de acordo com a interação do usuário.

Criei esse pequeno projeto para testar como manipular mais elementos de uma página HTML utilizando o JavaScript. Além de testar com mais eficácia como manipular os elementos HTML também aprendi como alterar propriedades CSS de modo a entender como tornar uma página web mais interativa! Alterar background, inserir ou modificar texto, inserir funções JavaScript utilizando eventos, como por exemplo, o evento 'onclick' que chama a função no arquivo JS externo e executa o que nós quisermos.
